# Netflix_Data_Analysis
Analyzed the Netflix Titles dataset using Python, SQL, and Pandas. Performed data cleaning, exploratory data analysis, and visualizations to identify trends in content types, genres, release years, ratings, countries, and directors.
Netflix Data Analysis

Project Overview

This project analyzes the Netflix Titles dataset using Python and SQL. The objective is to explore trends in movies and TV shows, clean the data, and generate insights through exploratory data analysis (EDA).

Objectives

- Clean and preprocess the dataset
- Analyze movies and TV shows
- Identify top content-producing countries
- Explore genres and content ratings
- Analyze release year and content addition trends
- Generate business insights using SQL and Python

Tools & Technologies

- Python
- Pandas
- Matplotlib
- SQL
- Git & GitHub

Project Files

- "netflix_titles.csv" – Dataset
- "analysis.py" – Python analysis script
- "sql_queries.sql" – SQL queries
- "README.md" – Project documentation

Key Insights

- Movies make up a larger share of Netflix's catalog than TV Shows.
- The United States has the highest number of titles.
- Drama and Comedy are among the most popular genres.
- TV-MA is one of the most common content ratings.
- Netflix significantly expanded its content library in recent years.

How to Run

1. Download the repository.
2. Install Python and the required libraries.
3. Run "analysis.py".
4. Review the generated analysis and visualizations.
